## Shared Message Pillars

Use these ideas consistently, but tune language to the channel.

- **Mature systems need continuity:** DAOhaus did not only need new features; it needed a safer operating model for live infrastructure.
- **AI readiness starts before automation:** codebase simplification, surface-area cleanup, and documentation made the AI-assisted workflow useful.
- **Human-in-the-loop is the point:** agents help with groundwork; humans approve production-affecting decisions.
- **Operational knowledge became durable:** docs, runbooks, access maps, and knowledge systems reduced dependence on historical contributors.
- **The raid became reusable infrastructure:** the engagement produced a repeatable Refactory pattern and service offering for future clients.
